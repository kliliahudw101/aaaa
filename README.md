# ZenithRecon
Advanced Cyber Intelligence & Reconnaissance System

## Description
ZenithRecon is a modular, AI-powered security reconnaissance framework designed for professional penetration testers. It integrates multiple LLMs via OpenRouter to provide deep analysis and collaborative exploitation strategies.

## Features
- **AI Collaboration:** Uses Stepfun, Nemotron, and GLM models for triage, analysis, and synthesis.
- **Modular Design:** Independent modules for DNS, Port Scanning, Vulnerabilities, and Recon.
- **Automated Tool Management:** Auto-installs and manages external security tools like XSStrike and Takeover.
- **Interactive Decision Making:** Human-in-the-loop for critical command execution.
- **Advanced Reporting:** Export results in JSON, CSV, and HTML.

## Installation
```bash
pip install -r requirements.txt
python zenith.py
```
